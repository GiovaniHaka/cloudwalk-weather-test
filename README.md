## API KEYS

1. Create an .env file on project root

2. Add following keys values to .env file:

```
OPEN_WEATHER_API_KEY=<value>
```

3. Clean flutter project:

```
flutter clean
```

3. Get all dependencies:

```
flutter pub get
```

3. Build:

```
flutter pub run build_runner build
```
